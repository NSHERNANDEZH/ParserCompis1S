# ParserCompis1S

Pasos para crear los archivos desde un archivo .g4 desde Intelliji IDEA
1. Abrir File -> Settings -> Plugins -> MarketPlace
2. Buscar antlr4 y descargar la del autor "TerrencePar"
3. Una vez instalado antlr, seleccione el archivo .g4 y configure antrl
4. Para configurarlo seleccione una carpeta donde se crearan los archivos (de preferencia la carpeta antl)
5. Una vez seleccionada la ruta, click derecho nuevamente al archivo .g4 y cree los archivos correspondiente
